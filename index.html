PROCESSO

Cada projeto é unico e tem suas necessidades específicas, mas essas são algumas
etapas comuns que eu sigo em meus projetos como boas práticas:

01

PESQUISA

Entender as metas e objetivos do projeto do cliente. Junto com o desenvolvimento de painéis de humor com base na visão do cliente.

02

CONCEITO

Use a pesquisa e desenvolva 2-3 conceitos aproximados. Em seguida, revise com o cliente para finalizar uma direção.

03

DESIGN

A fase de concepção e produção, onde todos os ativos são desenvolvidos ou prototipados para serem revisados pelo cliente.

04

REVIEW

Feedback de ida e volta com o cliente sobre alterações de design ou cópia antes da finalização dos ativos. Até 2-3 versões de provas.

05

FINALIZAÇÃO

Uma vez que o projeto seja aprovado, todos os ativos serão finalizados, os arquivos embalados e enviados para o cliente. O suporte contínuo é sempre uma opção.
